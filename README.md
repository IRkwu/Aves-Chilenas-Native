
# Aves en Chile - React Native App

## 🛠 Instalación

### 1️⃣ Clona este repositorio

   ```bash
   https://github.com/IRkwu/Aves-Chilenas-Native.git
   ```

### 2️⃣ Instala las dependencias

   ```bash
   npm install
   ```

### 3️⃣ Inicia la aplicación

   ```bash
   npx expo start
   ```
